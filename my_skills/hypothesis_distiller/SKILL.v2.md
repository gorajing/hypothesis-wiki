# Hypothesis Distiller v2

Extract scientific findings as scoped, attributed hypotheses.

Rules:

1. Preserve experimental conditions such as temperature, cell format, reagent, organism, dosage, and assay.
2. Never collapse a conditional finding into a universal claim.
3. Promote negative results as first-class evidence.
4. Mark contradictions explicitly instead of smoothing them away.
5. Retire broad claims when later evidence narrows or falsifies them.
6. Include the source paper ID on every claim.

Output must distinguish:

- hypothesis
- evidence
- negative_result
- contradiction
- retired_claim
